# Chatbot Upload Storage

Folder ini disediakan untuk menyimpan file chatbot yang ingin diunggah ke repository ini.

## Cara pakai
1. Simpan file chatbot kamu ke folder ini.
2. Gunakan nama file yang deskriptif, misalnya: `chatbot-v1.json`.
3. Commit perubahan agar file unggahan tersimpan di repository.

## Catatan
- Hindari mengunggah data sensitif (token, password, API key).
- Jika ukuran file besar, pertimbangkan kompresi terlebih dahulu.
